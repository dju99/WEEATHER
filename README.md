# WEEATHER

OpenAI와 날씨 API를 사용해 입을 옷을 추천 받을 수 있는 웹 애플리케이션입니다.

> 영문 지명으로 검색하면 현재 날씨와 3시간 간격으로 15시간 후 날씨 정보를 불러옵니다.

> 카드를 누르면 선택한 날씨를 기반으로 AI가 입을 옷을 추천해 줍니다.
